Graph Grad-CAM

A major criticism of deep learning in structural biology is the "black box" nature of predictions. It is not enough to know if two proteins interact; biologists must know how and where they interact to design drugs or mutagenesis experiments.

ProMaya utilizes a customized formulation of Gradient-weighted Class Activation Mapping (Grad-CAM), adapted specifically for Heterogeneous Graph Transformers.

Standard Grad-CAM vs. Graph Grad-CAM

Standard Grad-CAM was designed for 2D Convolutional Neural Networks processing images. It looks at the final 2D feature map and calculates the gradient of the target class with respect to that map.

ProMaya has no 2D feature map; it has a massive point cloud of node embeddings.

The ProMaya Formulation

We calculate the node-level Grad-CAM score directly on the embeddings generated at the final layer (Layer 4) of the HGT encoder.

  1. Calculate Gradients: Compute the gradient of the predicted interaction score (\(y^c\)) with respect to the \(k\)-th dimension of the final hidden embedding for node \(i\) (\(H_i^{(L),k}\)).
\[\alpha_i^k = \frac{\partial y^c}{\partial H_i^{(L),k}}\]
  1. Compute Salience: The importance (salience) of node \(i\) is the ReLU-activated sum of its embedding dimensions, weighted by the gradients.
\[\text{GradCAM}(i) = \text{ReLU} \left( \sum_k \alpha_i^k \cdot H_i^{(L),k} \right)\]

The ReLU activation ensures we only capture features that have a positive influence on predicting an interaction (ignoring features that suppress the prediction).

Code Example

src/interpretability/pymol_export.py
def _write_pymol_script(protein_a_path, protein_b_path, top_residues_a, top_residues_b, output_dir):
    """
    Generates an automated PyMOL script (.pml) to visualize interaction hotspots.
    The script loads both PDBs, colors them, and highlights the highest Grad-CAM scoring 
    residues as bright red and yellow sticks.
    """
    script_path = os.path.join(output_dir, 'interaction_view.pml')
    lines = [
        f'load {protein_a_path}, protA',
        f'load {protein_b_path}, protB',
        'hide everything',
        'show cartoon, protA or protB',
        'color marine, protA',
        'color orange, protB',
        'bg_color white',
        'set ray_opaque_background, off',
    ]

    # Highlight top Grad-CAM scoring residues in Protein A
    selection_a = ' or '.join([f"(protA and resi {row['resseq']})" for row in top_residues_a])
    lines.extend([
        f'select topA, {selection_a}',
        'show sticks, topA',
        'color red, topA',
    ])

    with open(script_path, 'w') as f:
        f.write('\n'.join(lines) + '\n')
    return script_path

Structural Visualization (PyMOL)

Rather than just returning abstract scores, ProMaya's prediction pipeline automatically writes a .pml script that can be directly executed by PyMOL (or imported into ChimeraX). This allows biologists to: - Instantly view the predicted binding interface in high-resolution 3D. - Inspect the top-scoring interaction pairs (visualized as dashed distance lines). - Identify actionable mutation targets or druggable allosteric pockets directly on the native structure.
